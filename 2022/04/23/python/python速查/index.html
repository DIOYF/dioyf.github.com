# logging 模块

import logging

日志格式规范，操作简化，日志分级管理，普通配置，复杂配置

日志级别：CRITICAL>ERROR>WARNING>INFO>DEBUG

功能 实现 参数 附加
将 debug 打印到标准输出 logging.debug(“debug message”)
将 info 打印到标准输出 logging.info(“info message”)
将 warning 打印到标准输出 logging.warning(“warning message”)
将 error 打印到标准输出 logging.error(“error message”)
将 critical 打印到标准输出 logging.critical(“critical message”)

“灵活配置日志级别，日志格式，输出位置”

类型功能 API 参数
logging.basicConfig () 设置日志级别 level = logging.DEBUG
输出格式，handler 的日志显示格式 format = '%(asctime)s % ’
指定日期格式 datefmt = ‘%a,%b, % Y’
指定文件名 filename = ‘test.log’
文件打开方式，默认为 “a” , 可指定为 "w" filemode = ‘a’

fh = logging.FileHandler (‘test1’, encoding = ‘utf-8’) 创建 handler 用于写入日志文件

sh= logging.streanHandler () 创建一个 handler 用于输出到控制台

formatter = logging.Formatter(’%(asctime)s-%(name)s-%(levelname)s -%(message)s ')

fh.setLevel (logging.DEBUG) 设置输出级别

sh.setLevel(logging.DEBUG)

# torchvision.transforms

常见的图像变换，这里总结一共有四大类，方便大家索引

裁剪（Crop）
中心裁剪：transforms.CenterCrop
随机裁剪：transforms.RandomCrop
随机长宽比裁剪：transforms.RandomResizedCrop
上下左右中心裁剪：transforms.FiveCrop
上下左右中心裁剪后翻转，transforms.TenCrop

翻转和旋转（Flip and Rotation）
依概率 p 水平翻转：transforms.RandomHorizontalFlip (p=0.5)
依概率 p 垂直翻转：transforms.RandomVerticalFlip (p=0.5)
随机旋转：transforms.RandomRotation

图像变换（resize）transforms.Resize
标准化：transforms.Normalize
转为 tensor，并归一化至 [0-1]：transforms.ToTensor
填充：transforms.Pad
修改亮度、对比度和饱和度：transforms.ColorJitter
转灰度图：transforms.Grayscale
线性变换：transforms.LinearTransformation ()
仿射变换：transforms.RandomAffine
依概率 p 转为灰度图：transforms.RandomGrayscale
将数据转换为 PILImage：transforms.ToPILImage
将 lambda 应用作为变换：transforms.Lambda

对 transforms 操作，使数据增强更灵活
从给定的一系列 transforms 中选一个进行操作：transforms.RandomChoice (transforms)，
给一个 transform 加上概率，依概率进行操作 ：transforms.RandomApply (transforms, p=0.5)
将 transforms 中的操作随机打乱：transforms.RandomOrder
