# 电子计算机，基本硬件单元

名词 概念
P1,P2: 电子计算机 讲计算机发展史 Electronic Computing
P3: 布尔逻辑 Boolean Logic 与，或，非，异或，同或
P3: 逻辑门 Logic Gates 由对应逻辑组成的（输入输出）单元
P4: 二进制 Binary 0，1 两种状态进行存储
P5: 算术逻辑单元 ALU 由逻辑门构成的计算单元（数字电路）
P5 :ALU 包含： 算数单元，状态标志位。
P5: ALU 可以组成 CPU 中央处理器，含有很多处理单元。
P6: 寄存器 Registers， 从数字电路设计寄存器，实现锁存
P6: 内存 Memory 寄存器集成，可以组成大的 Memory。
P6: 锁存 RAM RAM 行线列线结构，两根线传递。
P6: 多路复用器，译码器 如今有 SRAM，DRAM 等等一些列抽象
P7: CPU 给 CPU 指令存储数据
P7: 微机系统 取指令，取地址 RAM，读数据送入 ALU，输出结果
P7: 超频，降频 调整时钟频率，低功耗或者高频处理
P8: 微机原理，加指令集 ALU+RAM+Control Unit 使用指令集操作
P9: 高级 GPU 增加主频，增加计算能力
P9: Pipeline 流水线处理 流水线技术进行处理，可以并行处理
P9: 动态排序 使用复杂电路进行动态排序
P9: 超级计算机 多处理器，巨型处理器

名词 概念
P10: 打卡 打卡式计算机

P10: 早期编程 汇编语言：二进制

P11: COBOL 一个通用的编写语言，一次编写到处运行
P11: 编程应用 计算机从深奥学科变成大众化工具。
P11：编程语言 C++, C#, python
P12: 通用元素 for 循环，while 循环，条件判断语句
P12: 函数 代码打包，函数，方法，子方法
P13: 算法 从简单算法到人工智能算法
P14： index 下标和索引
P14： 数据结构 Array-List-Vector （向量数据结构）
P14: 数组 下标从 0 开始计算
P14: 字符串 字符串结构，注意结尾字符 \n
P14: 高维数组 在内存里线性排列的数组
P14：链表，队列 循环链表，非循环链表，队列结构（que）
P14：栈 后进先出（push and pop）
P14: 图（graph）
P14：树（tree）
P14：其他结构 红黑树，堆
P14：标准库 C++ 有 STL，java 有 java 类库
P15: 阿兰图灵 阿兰图灵（图灵天才） 图灵机 tuing machine

P33 : 现代存储 硬盘，软盘，CD，固态（U 盘）。SSD 固态

P23：CAD 计算机辅助设计，早期 CAD 软件
